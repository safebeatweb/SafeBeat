The Project

Protecting Every Beat.

SafeBeat is a comprehensive STEM research project focused on detecting cardiovascular stress in real-time. By utilizing high-fidelity sensors and AI, we bridge the gap between training and clinical safety.

Section 2

Integrating Science, Technology, Engineering, and Mathematics to build a practical solution for sudden cardiac events in sports.

Hardware Eng.

High-frequency signal processing using the AD8232 frontend to extract clean P-QRS-T waveforms.

Data Sync

Encrypted transmission to Firebase Europe nodes ensuring real-time telemetry across all mission command units.

The Intelligence

Precision Logic.

SafeBeat uses a personalized threshold algorithm: Threshold = (220 - Age) × ActivityFactor. This allows for clinical-grade precision tailored to the individual athlete's biological profile.

Signal Processing

Digital filtering algorithms remove 50/60Hz power line interference to reveal pure cardiac biopotential.
